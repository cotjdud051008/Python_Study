## 3. 변수로 연산하기

연산을 쉽게 하기 위해 변수에 연산 결과를 저장한 뒤 새로운 연산에 활용할 수 있습니다. 이때 복합 대입 연산자를 사용하면 더욱 간소한 형태로 연산이 가능합니다.
